GitHub Setup for Dance Silhouette Booth

This guide will help you set up your application to save silhouettes directly to your GitHub repository.

Important: Your personal access token grants access to your GitHub account. Never share it with others or commit it to public repositories.

Step 1: Create a GitHub Repository

  1. Go to github.com/new
  2. Name your repository (e.g., "dance-silhouettes")
  3. Make it public or private according to your preference
  4. Click "Create repository"

Step 2: Generate a Personal Access Token

  1. Go to github.com/settings/tokens
  2. Click "Generate new token" (classic)
  3. Name your token (e.g., "Dance Silhouette Booth")
  4. Set expiration according to your preference
  5. Select the "repo" scope (allows managing repositories)
  6. Click "Generate token"
  7. Copy your token immediately! You won't be able to see it again.

Step 3: Configure Your Application

  1. Open the file sketch.js in a code editor
  2. Find the following section near the beginning of the downloadSilhouette function:
    3. // GitHub settings - REPLACE THESE WITH YOUR OWN VALUES const GITHUB_TOKEN = ""; // ADD YOUR TOKEN HERE const GITHUB_USERNAME = ""; // ADD YOUR USERNAME HERE const GITHUB_REPO = ""; // ADD YOUR REPOSITORY NAME HERE const GITHUB_BRANCH = "main";
  3. Fill in your GitHub username, repository name, and the token you generated
  4. Save the file

Step 4: Test Your Setup

  1. Return to the main application
  2. Record and save a silhouette
  3. You should see your files being uploaded to GitHub
  4. You'll get a link to view them on GitHub when finished
